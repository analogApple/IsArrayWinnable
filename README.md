# IsArrayWinnable

IsArrayWinnable is a React Native project for job interview

## Installation & Usage

first read setting up the development environment for React Native:
https://reactnative.dev/docs/environment-setup

use git to clone repository

```bash
git clone https://github.com/analogApple/IsArrayWinnable.git
```

open project directory

```bash
cd IsArrayWinnable
```

then install project dependencies

```bash
yarn
```

or

```bash
npm install
```

run on iOS simulator

```python
react-native run-ios
```

run on Android virtual or real device

```python
react-native run-android
```
